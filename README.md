# 基金净值自动更新器 (Fund Value Updater)

一个基于 Python + AKShare 的**自动化基金数据追踪工具**，可将你关注的公募基金、ETF 净值及多维度涨跌幅，每日自动更新并写入本地 Excel 文件，方便长期追踪与管理。

---

## 功能特性

| 功能 | 说明 |
|---|---|
| **多维度指标** | 自动计算当日、近7天、近1月、近1年涨跌幅，及成立以来收益率 |
| **分类管理** | Excel 中包含「主推公募」「ETF」「个人关注产品」「私募资管」共4个 Sheet |
| **历史文件继承** | 新的一天运行时，自动复制最近历史文件内容（包括基金列表），无需重复配置 |
| **私募资管保护** | 私募资管 Sheet 完全不自动处理，支持用户自由定制内容和格式 |
| **收益率精确计算** | 成立以来收益率优先使用**累计净值**（含分红复利）；短期涨跌幅使用**单位净值** |
| **建仓期基金支持** | 对封闭期或净值未披露的基金，仍能显示基金名称并标记状态 |
| **格式自动优化** | 列宽自动调整（适配长中文名），净值5位小数，涨跌幅2位小数+%，A列文本格式 |
| **反风控保护** | 请求间隔随机化（0.3~0.6秒），避免触发 API 限流 |

## 环境要求

- Python 3.8+
- 依赖库（运行以下命令安装）：

```bash
pip install akshare pandas openpyxl
```

## 快速开始

### 1. 填入基金代码

打开 `fund_data/` 目录中的 Excel 文件（或首次运行先执行脚本生成）。

在 **「主推公募」「ETF」「个人关注产品」** 三个 Sheet 的 **A 列**（从第 2 行开始），手动填入你关注的基金代码（6位数字，如 `000001`）。

### 2. 运行脚本

- **双击** `run_fund_update.bat`（最简单）
- 或在命令行中运行：

```bash
python main.py
```

### 3. 查看结果

脚本运行完毕后，`fund_data/` 目录中会生成（或更新）当天的 Excel 文件：`fund_tracker_YYYY-MM-DD.xlsx`。

## Excel 表结构

```
A列: 产品代码
B列: 产品名称（自动填充）
C列: 最新净值
D列: 当日涨跌幅(%)
E列: 近7天涨跌幅(%)
F列: 近1月涨跌幅(%)
G列: 近1年涨跌幅(%)
H列: 成立以来收益率(%)
J1:  更新时间（自动写入）
```

> **私募资管 Sheet** 不会自动更新，所有内容由用户手动维护，格式完全自定义。

## 自动化定时运行（可选）

可通过 Windows 任务计划程序实现每天 16:00 自动运行：

1. 按 `Win + R`，输入 `taskschd.msc` 打开任务计划程序。
2. 点击「创建基本任务」：
   - **触发器**：每天 → 启动时间 `16:00:00`
   - **操作**：启动程序 → 选择 `run_fund_update.bat`
   - **起始于**：填写项目所在目录路径（如 `F:\Project\fund.value.updater`）
3. 点击「完成」。

> 只要电脑在 16:00 开机，任务将自动执行并更新 Excel 文件。

---

## 注意事项

- **同一天多次运行**：会覆盖同一个文件中的数据（更新时间同步刷新），不会生成重复文件。
- **跨日继承**：第二天运行时，自动从最近的历史文件复制基金列表，无需重新填写代码。
- **网络依赖**：数据来源为 AKShare（东方财富接口），请确保运行时可以正常访问互联网。
