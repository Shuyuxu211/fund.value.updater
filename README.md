# 基金净值自动更新器 (Fund Value Updater)

一个基于 Python + AKShare 的**自动化基金数据追踪工具**，以**累计净值**为统一数据源，自动计算各周期收益率、近1年波动率和夏普比率，并写入本地 Excel 文件。

---

## 功能特性

| 功能 | 说明 |
|---|---|
| **真实复权净值** | 舍弃失真的累计净值，利用官方除权后的“日增长率”按日执行几何累乘，凭空重构绝对真实的归一化分红再投资“复权净值曲线” |
| **多维度收益率** | 自动计算当日、近7天、近1月、近1年收益率及成立以来收益率 |
| **风险指标** | 近1年年化波动率（日收益率标准差×√252）和夏普比率（无风险利率1.5%），无额外请求 |
| **自动排序** | 每次运行后各Sheet按当日收益率降序排列，None值置底 |
| **分类管理** | 自动处理「主推公募」「ETF」「个人关注产品」3个Sheet；「私募资管」完全不触碰，保留手动内容 |
| **历史文件继承** | 新的一天运行时，自动复制最近历史文件内容，无需重复配置 |
| **建仓期基金支持** | 对封闭期或净值未披露的基金，仍能显示基金名称并标记状态 |
| **格式自动优化** | 列宽自动调整（适配长中文名），净值5位小数，收益率/波动率2位小数+%，夏普2位小数 |
| **反风控保护** | 请求间隔随机化（0.3~0.6秒），避免触发 API 限流 |

```bash
pip install akshare pandas openpyxl
```

## 快速开始

### 1. 填入基金代码

打开 `fund_data/` 目录中的 Excel 文件（首次运行先执行脚本生成）。

在 **「主推公募」「ETF」「个人关注产品」** 三个 Sheet 的 **A 列**（从第 2 行开始），手动填入你关注的基金代码（6位数字，如 `000001`）。

### 2. 运行脚本

- **双击** `run_fund_update.bat`（最简单）
- 或在命令行中运行：

```bash
python main.py
```

### 3. 查看结果

脚本运行完毕后，`fund_data/` 目录中会生成（或更新）当天的 Excel 文件 `fund_tracker_YYYY-MM-DD.xlsx`，数据行已按当日收益率降序排列。

## Excel 表结构（自动处理的3个Sheet）

```
A列: 产品代码
B列: 产品名称（自动填充）
C列: 最新单位净值
D列: 当日收益率(%)
E列: 近7天收益率(%)
F列: 近1月收益率(%)
G列: 近1年收益率(%)
H列: 近1年波动率(%)        ← 年化标准差，衡量波动风险
I列: 近1年夏普              ← 无量纲，越高越好
J列: 成立以来收益率(%)
K1:  更新时间（自动写入）
```

> **计算说明（真实复利模型）**
> - **最新单位净值 (C列)**：严格展示官方当天公布的实时当前持仓净值。
> - **区间收益率 (D-G, J列)**：为了抹平大比例分红和份额拆分导致的断层，程序将提取基金成立以来的所有“日增长率”，从 `1.0` 几何连乘 (`cumprod`) 重构出一条绝对真实的 **归一化复权净值曲线**。所有长短期收益率均基于该合成曲线计算，完美呈现“分红再投资”的复利滚雪球效应。
> - **波动率**：近 252 个交易日的真实 **日增长率** 的标准差 × √252（年化）
> - **夏普比率**：`(近1年合成复权收益率 - 1.5%) / 年化波动率`，无风险利率默认 1.5%
> - **数据行排序**：每次运行后按D列（当日收益率）降序，无数据的基金置底
>
> **私募资管 Sheet** 不会自动更新，所有内容由用户手动维护，格式完全自定义。

## 自动化定时运行（可选）

通过 Windows 任务计划程序实现每天 16:00 自动运行：

1. 按 `Win + R`，输入 `taskschd.msc` 打开任务计划程序。
2. 点击「创建基本任务」：
   - **触发器**：每天 → 启动时间 `16:00:00`
   - **操作**：启动程序 → 选择 `run_fund_update.bat`
   - **起始于**：填写项目所在目录路径（如 `F:\Project\fund.value.updater`）
3. 点击「完成」。

---

## 注意事项

- **同一天多次运行**：会覆盖同一个文件中的数据（时间戳同步刷新），不会生成重复文件。
- **跨日继承**：第二天运行时，自动从最近的历史文件复制基金列表，无需重新填写代码。
- **网络依赖**：数据来源为 AKShare（东方财富接口），请确保运行时可以正常访问互联网。
- **波动率/夏普**：仅在基金成立满1年后才计算并显示；近1年收益率为负时，夏普比率列留空（负收益率下夏普无参考意义）。
